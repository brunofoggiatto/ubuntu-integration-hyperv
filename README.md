# ubuntu-integration-hyperv
Em clusters Windows Server, a integração de VMs Linux com o Hyper-V depende do suporte do kernel aos módulos e daemons (KVP, VSS e FCopy). Algumas distribuições não incluem esses componentes, gerando falhas de coleta e integração. Esta solução adiciona os scripts e ferramentas necessários, restaurando a funcionalidade completa.
